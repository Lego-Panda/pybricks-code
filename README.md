# Lego Panda 2025-2026 Pybricks Code

## Functions:
 - pid: move straight with PID
 - decel: include deceleration with PID
 - turn: turn with PID
 - shellTurn: turn shell with PID
 - turnWhileShell: turn robot while turn shell
 - stopColor: stop shell using the color sensor
 - shellButton: stop shell when clicking the buttons
